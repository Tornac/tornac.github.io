Tornac's Github Pages

This site serves as a landing page for some sample web apps I've created in the past. The apps are generally fairly simple and were created as a learning experience and not originally intended to be published anywhere.

Link to Repository

The repository itself can be found at https://github.com/Tornac/tornac.github.io and contains the source code for the projects presented here.

Maze Game

Link to website: https://tornac.github.io/maze2

A simple game where the player can navigate a maze. The game uses Typescript, but no web frameworks.

Todo-List

Link to website: https://tornac.github.io/react-todo

An application allowing the user to manage a todo-list, allowing the creation of new elements as well as their deletion. Entries in the todo-list can be marked as completed and they are stored on the device they are created on. This project uses React and Typescript.

ffxiv-cooldowns

Link to website: https://tornac.github.io/ffxiv-cooldowns

Created as a joke, referencing a common phrase in a video game. Features some very simple CSS animations, but server no purpose otherwise.
